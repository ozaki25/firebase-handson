## 登場する技術

### Firebase

- AWSのようにさまざまなサービスを提供するクラウドサービス
- 今回はSNS連携による認証機能を提供するFirebase Authenticationを使ってみます
- 後ろだてはGoogle

### React

- SPAフレームワーク
- クライアントサイドのフレームワークなのでブラウザ上で実行され実装はJavaScriptで行う
- コンポーネント(≒HTMLの塊)を作成し組み合わせることで開発していく
